# Программа для мониторинга курсов валют
Данная программа выводит в консоль курсы валют из exchangerate-api в следующем формате.
```
USD Доллар      74.96251874062969
EUR Евро        87.87346221441125
Введите 1 - Обновить данные | 2 - Редактировать список валют | 0 - Выйти
```
Пользователь имеет возможность редактировать список выводимых валют, добавляя их по индексу из трех букв а также давать им собственные названия.
```
Тег: USD        Название: Доллар
Тег: EUR        Название: Евро
Введите 1 - Для добавления/редактирования валюты | 2 - Для удаления валюты | 0 - Для выхода
```
Если пользователь вручную обновляет данные, то перед запросом система проверит дату последнего обновления и список запрошенных в последний раз валют. Если список валют отличается от запрошенного в последний раз 
или время запроса превышает дату следующего обновления полученную из сервера, то данные будут запрошены повторно. В противном случае пользователю будут выведены данные предыдущего запроса.
